# Stockholm

## 🛡 Educational Ransomware

This project simulates a simple ransomware program (for educational purposes only).  
It can encrypt and decrypt files in a specified directory, and includes command-line options for different modes.

## 📚 Features
- Encrypt / decrypt files using AES or similar secure algorithm.
- Command-line flags:
  - `--help` : display usage.
  - `--version` : show version.
  - `--silent` : suppress console output.
  - `--reverse` : decrypt mode.

## ⚙️ Build & Run
```bash
make
./stockholm --help
